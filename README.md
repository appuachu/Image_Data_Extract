# Image_Data_Extract
This tool is used to Extract Data from Image.It is a shell script to Automate the process



<h1>Installation</h1>
$ git clone https://github.com/appuachu/Image_Data_Extract.git<br>
$ cd Image_Data_Extract<br>
$ chmod +x *<br>
$ bash install.sh<br>
$ bash imagedata.sh
